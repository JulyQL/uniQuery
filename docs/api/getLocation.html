getLocation 获取地理位置
H5 微信小程序 支付宝小程序
是否支持？ ✓ ✓ ✓

使用 

uni.$uniQuery.getLocation((res) => {
    // todo
});

其中，res的数据格式如下： 

{
    longitude: number // 经度
    latitude: number // 纬度
}

权限注册

部分端需要提前在 manifest.json 中进行权限申请的注册：

微信小程序 

"mp-weixin": {
    "permission": {
        "scope.userLocation": {
            "desc": "您的位置信息将用于小程序页面"
        }
    },
    "requiredPrivateInfos": [
        "getLocation"
    ]
}
